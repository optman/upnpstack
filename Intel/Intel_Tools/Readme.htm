What new in the release?

  • AV Control Points (AV Wizard and AV Controller) now generate Extended M3U play lists. New menu options to turn this feature off. Note that the AV Renderer uses the Media Player ActiveX control that does not support Extended-M3U.
  • New debugging interface for all the tools. This is generally located in the Help menu of the tools.
  • Fixed many problems: AV Renderer event escaping, handling invalid SID events and much more.
  • PrepareForConnection is now removed by default in the AV Renderer.
  • Added a new PocketPC Media Renderer tool in the "Micro Tools" directory.
  • New help file attached to all of the Tools.

Intel Device Spy - is the premier Universal Plug & Play Universal Control Point (UCP). It allows UPnP device detection, display of detailed information on every aspect of a UPnP device, action invocation, event monitoring and error handling. The application is built with a solid, easy to use, user interface and leverages Intel's Universal Plug & Play CLR stack.

Intel Service Author - is an easy to use SCPD XML authoring tool. Creating and maintaining UPnP services is now extremely simple, our SCDP parser will also generate and validate the document for compliance. Service Author includes a C# code generator, this feature is not useful outside of Intel.

Intel Device Sniffer - monitors all UPnP broadcast traffic on the network. Solving SSDP interoperation problems has never been easier. The application can filter packets and issue SSDP searches, very useful for plug fests events.

Intel Device Validator - is a battery of tests that can be applied to any UPnP device. Validator includes tests that apply to all devices, and AV specific tests for AV Renderers and AV Content Directory Servers (CDS).

Intel Device Scriptor - an easy to use UPnP scripting application. Allows the user to quickly build scenario scripts and run them against UPnP devices.

Intel AV Media Controller - is a fully features UPnP AV Control Point. It is equivalent to Device Spy for AV. AV Media Controller looks for all renderers and content directories on the network, it virtualizes all of the media content for rapid searching, dynamically builds play lists for renderers that support it, supports single and multi-instance renderers and much more. This is the must have, UPnP AV application.

Additional notes on UPnP Media Controller:

  • Right click on a renderer and select "Renderer Controls" for more renderer options.

Intel AV Media Server - is complete UPnP AV Content Directory. It uses the file system as a database allowing the user to drag & drop folders for quick sharing of media. Media Server's user interface also allows tracking of downloaded files.

Additional notes on  AV Media Server:

  • You can drag & drop folders onto the user interface or use the "add shared folder" menu option.
  • MP3 ID3 version 1 tags, GIF and JPEG metadata is supported. Other file types will show up with generic metadata.

Intel AV Media Renderer - a fully features UPnP AV renderer. Supports multi-instance, play lists support, complete moderated eventing, track seeking support, position seeking support and much more.

Additional notes on AV Media Renderer:

  • Run AV Media Renderer and select the features and MIME types the renderer will support. Once everything is setup, select "Start Renderer" from the File menu.
  • Many UPnP properties including NextContentURI and many display attributes don't have any effect, but are left for testing purposes.

Intel AV Wizard - is an UPnP AV control point for playing locally stored content to available AV renderers. The goal if to make this application integrate well within Windows XP as a shell extension. Select a renderer and drag & drop a file into AV Wizard to start playing.

Intel Network Light - Also included in the package is a simple UPnP home automation light that can be used to demonstrate the basic abilities of both Device Spy and Service Author.

Intel Device Relay - Mirrors UPnP Device onto a different network. Start Device Relay on two different computers and use a WAN interface to connect one to the other. UPnP Relay will automatically mirror all other devices from one network on to the other. Relayed devices will appear with a star (*) in front of their names. SSDP messages are handled locally by the relay, events and action invocations are forwarded to the source relay using SOAP.

Internationalization

All tools should work with large fonts and with any display DPI settings. Please report any problems.

Intel Device Spy and Network Light as been localized to French. It will run in French on a French computer. Users can override the language setting with the "-fr" (French) and "-en" (English) command line parameter.
